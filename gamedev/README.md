# Изучение закономерностей, определяющих успешность игр

## Данные

Из открытых источников доступны нижеприведенные исторические данные о продажах игр, оценки пользователей и экспертов, жанры и платформы (например, Xbox или PlayStation):
- название игры
- платформа
- год выпуска
- жанр игры
- продажи в Северной Америке, Европе, Японии и в других странах (миллионы проданных копий)
- оценка критиков (максимум 100)
- оценка пользователей (максимум 10)
- рейтинг от организации ESRB (англ. Entertainment Software Rating Board). Эта ассоциация определяет рейтинг компьютерных игр и присваивает им подходящую возрастную категорию.

## Задача
Используя исторические данные о продажах компьютерных игр, оценки пользователей и экспертов, жанры и платформы, выявить закономерности, определяющие успешность игры.

## Используемые библиотеки

*pandas, seaborn, matplotlib, scipy.stats*
